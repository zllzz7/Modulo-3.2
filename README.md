# Módulo 3.2 – Página Web Informativa sobre Desarrollo Web

## Descripción
Página web informativa creada como entrega individual del Módulo 3.2.  
Abarca los conceptos fundamentales de **HTML**, **CSS** y **Git**.

## Archivos del proyecto

| Archivo      | Descripción                                      |
|--------------|--------------------------------------------------|
| `index.html` | Estructura principal de la página con menú de navegación |
| `style.css`  | Estilos visuales, layout responsivo y animaciones |
| `README.md`  | Documentación del proyecto (este archivo)        |

## Estructura de la página

- **Inicio (Hero)** – Presentación del tema con título y llamada a la acción  
- **HTML** – Tarjetas sobre estructura, semántica y accesibilidad + ejemplo de código  
- **CSS** – Explicación de selectores, propiedades y ejemplo de código  
- **Git** – Línea de tiempo con los comandos básicos: `init`, `add`, `commit`, `push`  
- **Contacto** – Formulario funcional con validación básica  

## Tecnologías utilizadas

- HTML5 semántico  
- CSS3 (Variables, Flexbox, Grid, animaciones, media queries)  
- JavaScript vanilla (menú responsive, IntersectionObserver, formulario)  
- Google Fonts: *Syne* + *DM Sans*
