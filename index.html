01

¿Qué es HTML?

📄

Estructura

HTML (HyperText Markup Language) es el esqueleto de toda página web. Define el contenido y su organización mediante etiquetas.

🏷️

Semántica

Las etiquetas semánticas como <header>, <section> y <footer> dan significado al contenido y mejoran el SEO.

♿

Accesibilidad

Un HTML bien escrito usa atributos alt, aria-label y estructura jerárquica para que todos puedan usar la web.

Ejemplo básico de HTML
<!-- Estructura mínima de una página -->
<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8"/>
    <title>Mi Página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
  </body>
</html>
02

¿Qué es CSS?

CSS (Cascading Style Sheets) controla la presentación visual de una página web. Trabaja junto a HTML para definir colores, tipografía, espaciado y diseño.

  • 🎨 Colores y tipografía
  • 📐 Modelo de caja (Box Model)
  • 🔲 Flexbox y Grid para layouts
  • 📱 Diseño responsivo con Media Queries
  • ✨ Animaciones y transiciones
Ejemplo básico de CSS
/* Variables de color */
:root {
  --primario: #0a84ff;
  --fondo:   #0d0d0d;
}

/* Estilo del encabezado */
header {
  background: var(--fondo);
  color:      #fff;
  padding:    1rem 2rem;
}

Control de versiones con Git

Git permite registrar cada cambio en el código, trabajar en equipo sin conflictos y mantener un historial completo del proyecto.

1

git init

Inicializa un repositorio vacío en la carpeta del proyecto.

git init
2

git add

Agrega archivos al área de preparación (staging) antes del commit.

git add .
3

git commit

Guarda una instantánea del estado actual con un mensaje descriptivo.

git commit -m "feat: agregar index.html"
4

git push

Sube los commits locales al repositorio remoto (GitHub, GitLab, etc.).

git push origin main
