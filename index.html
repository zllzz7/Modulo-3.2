01

¿Qué es HTML?

📄

Estructura

HTML es el esqueleto de toda página web. Define el contenido y su organización mediante etiquetas.

🏷️

Semántica

Las etiquetas semánticas dan significado al contenido y mejoran el SEO de manera profesional.

♿

Accesibilidad

Un HTML bien escrito permite que todos los usuarios puedan navegar la web sin barreras.

Ejemplo básico de HTML
<!-- Estructura mínima -->
<!DOCTYPE html>
<html lang="es">
  <body>
    <h1>¡Hola Mundo!</h1>
  </body>
</html>
02

¿Qué es CSS?

CSS controla la presentación visual. Trabaja junto a HTML para definir colores, tipografía y diseños responsivos.

  • 🎨 Colores y tipografía
  • 📐 Modelo de caja (Box Model)
  • 🔲 Flexbox y Grid Layouts
  • ✨ Animaciones y transiciones
Ejemplo básico de CSS
:root {
  --primario: #00c6ff;
  --fondo:   #001f8f;
}

header {
  background: var(--fondo);
  backdrop-filter: blur(10px);
}
03

Control de versiones con Git

Git permite registrar cada cambio en el código y trabajar en equipo de forma eficiente.

1

git init

Inicializa un repositorio en la carpeta del proyecto.

git init
2

git add

Prepara los archivos para el guardado.

git add .
3

git commit

Guarda una instantánea con un mensaje descriptivo.

git commit -m "feat: new hero"
